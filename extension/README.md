# HTML转Markdown 浏览器扩展

将HTML代码或网页内容转换为Markdown格式的Chrome/Edge浏览器扩展。

## 安装方法

1. 打开 Chrome 或 Edge 浏览器
2. 地址栏输入 `chrome://extensions/`（Edge 输入 `edge://extensions/`）
3. 打开右上角 **"开发者模式"** 开关
4. 点击 **"加载已解压的扩展程序"**
5. 选择本文件夹（`extension`）
6. 工具栏出现紫色 **H2M** 图标即安装成功

## 使用方式

### 方式一：粘贴HTML代码

1. 点击工具栏的扩展图标，弹出窗口
2. 在输入框中粘贴HTML代码
3. 点击 **"转换"** 按钮（或按 `Ctrl+Enter`）
4. 下方显示Markdown源码和预览

### 方式二：转换当前网页

1. 打开任意网页
2. 点击扩展图标 → 切换到 **"转换当前页面"** 标签
3. 点击按钮，自动提取页面主体内容并转换
4. 可复制或下载为 `.md` 文件

### 方式三：转换选中内容

1. 在网页上用鼠标选中一段文字
2. 点击扩展图标 → 切换到 **"转换选中内容"** 标签
3. 点击按钮，仅转换选中的部分

### 方式四：右键菜单

在任意网页上右键点击，菜单中会出现：

- **将此页面转换为Markdown** — 提取整个页面内容，弹出侧边结果面板
- **将选中内容转换为Markdown** — 先选中文字，再右键转换

结果面板支持复制、下载、按 `ESC` 关闭。

### 方式五：拖拽文件

1. 点击扩展图标打开弹窗
2. 切换到 **"粘贴HTML"** 标签
3. 直接把 `.html` 或 `.htm` 文件拖入输入框
4. 文件内容自动加载，点击转换即可

## 预览功能

转换完成后，输出区域有两个标签：

- **源码** — 显示原始Markdown文本
- **预览** — 渲染后的效果（标题、列表、表格、代码块等）

## 配置转换规则

点击弹窗右上角的 **⚙** 按钮，进入设置页面，可配置：

| 选项 | 可选值 | 说明 |
|------|--------|------|
| 标题风格 | ATX / Setext | `# 标题` vs `标题\n====` |
| 列表标记 | `-` / `*` / `+` | 无序列表前缀 |
| 代码块 | 围栏 / 缩进 | \`\`\` vs 4空格 |
| 分隔线 | `---` / `***` / `___` | 水平线样式 |
| 斜体 | `*` / `_` | `*文本*` vs `_文本_` |
| 粗体 | `**` / `__` | `**文本**` vs `__文本__` |
| 链接 | 内联 / 引用 | `[t](u)` vs `[t][1]` |

修改后点 **"保存设置"**，设置通过Chrome账号跨设备同步。

## 快捷键

| 快捷键 | 作用 |
|--------|------|
| `Ctrl+Enter` | 在输入框中快速执行转换 |
| `ESC` | 关闭右键菜单的结果面板 |

## 常见问题

**Q: 点击"转换当前页面"没反应？**

A: 浏览器内部页面（`chrome://`、`edge://`、`about:` 等）无法使用此功能，请在普通网页上操作。

**Q: 右键菜单没有出现？**

A: 确认扩展已启用。刷新页面后重试。

**Q: 预览效果和源码不一致？**

A: 预览使用内置简易渲染器，复杂Markdown以源码为准。复制源码到专业编辑器（如VS Code、Typora）可获得完整渲染。

## 技术栈

- [Turndown.js](https://github.com/mixmark-io/turndown) — HTML to Markdown 转换引擎
- [turndown-plugin-gfm](https://github.com/mixmark-io/turndown-plugin-gfm) — GFM扩展（表格、删除线、任务列表）
- Chrome Extension Manifest V3
